Jaký přístup používá Bulma?

Bulma je komponentový CSS framework s prvky utility přístupu.

Komponentový

  • Předpřipravené komponenty (Button, Card, Navbar...)
  • Konzistentní vzhled
  • Sémantické názvy tříd

Utility helpery

  • Třídy pro spacing: m-4, p-2
  • Barvy: has-text-primary
  • Flexbox: is-flex

Jak načíst Bulmu do projektu

1. CDN (nejjednodušší) 

<link rel="stylesheet" 
  href="https://cdn.jsdelivr.net
  /npm/bulma@1.0.2/css/bulma.min.css">

Stačí jeden řádek v <head>

2. NPM instalace 

npm install bulma

// V SASS/CSS souboru:
@import "bulma/css/bulma.css";

Pro customizaci a build systémy

Bulma je čistě CSS - nepotřebuje JavaScript!

Systém modifikátorů

Prefix is-

Pro stavy a velikosti:

  • is-primary - primární barva
  • is-large - velká velikost
  • is-loading - načítání
  • is-active - aktivní stav

Prefix has-

Pro vlastnosti:

  • has-text-centered
  • has-background-dark
  • has-text-weight-bold
  • has-shadow

Třídy jsou snadno zapamatovatelné a logické!

Kdy použít Bulmu?

Ideální pro:

  • Rychlé prototypy - rychlý start bez konfigurace
  • Menší až střední projekty - blogy, portfolia, landing pages
  • Projekty bez JS frameworku - čisté HTML/CSS
  • Začátečníky - snadná syntaxe
  • Responzivní weby - vestavěný mobilní grid

Výhody:

  • Žádný JavaScript - čistě CSS
  • Modulární - import jen potřebných částí
  • Dobrá dokumentace
  • Snadná customizace přes SASS proměnné
  • Malá velikost (~200KB)
6 / 8

Nevýhody a omezení

Omezení

  • Žádné JS komponenty - modály, dropdowny vyžadují vlastní JS
  • Menší komunita než Bootstrap nebo Tailwind
  • Méně utility tříd než Tailwind CSS
  • Omezené animace - nejsou součástí

Kdy nepoužívat

  • Velké enterprise aplikace
  • Projekty vyžadující hodně interaktivity
  • Když potřebujete maximální kontrolu nad styly
  • Projekty s unikátním designem

Srovnání: Bootstrap má více funkcí, Tailwind více flexibility, ale Bulma je jednodušší na naučení.
